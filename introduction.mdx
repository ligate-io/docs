Everything you need to build on the Ligate sovereign rollup. The attestation
protocol, Proof of Prompt receipts, the Mneme wallet surface, the Iris MCP
server, client SDKs, node operator guides, and the reference for every
primitive.

<Tip>
  `ligate-devnet-1` is live on Celestia Mocha-testnet. The CLI, faucet, RPC, and
  explorer are all running. v1 hardening continues on top, breaking changes
  flagged in the [changelog](https://ligate.io/changelog).
</Tip>

This is the developer documentation for Ligate Labs. It is early. Some pages
are drafts, some are empty outlines. We're filling it in as the devnet firms
up — the goal is that by mainnet, every surface a developer touches is
documented, with runnable examples.

## By product

<CardGroup cols={3}>
  <Card title="Themisra" icon="check-double" href="/themisra">
    Hash any prompt + output pair, quorum-attest it on-chain, verify it
    anywhere. Model-agnostic (Claude, GPT, Gemini, Llama, any OpenAI-compatible
    endpoint).
  </Card>

<Card title="Mneme" icon="wallet" href="/mneme">
  First-party non-custodial browser wallet for Ligate. ed25519 native, signs
  every chain transaction, ships with a one-click "Attest with Mneme" button
  injected into ChatGPT, Claude, and Gemini.
</Card>

  <Card title="Iris" icon="robot" href="/iris">
    MCP server + USD-billed relayer. Any Claude Desktop, Cursor, LangGraph, or
    OpenAI Agents SDK agent gets cryptographic attestation of every action in
    ~10 lines.
  </Card>
</CardGroup>
