<Info>
  This page is in active migration. The TypeScript SDK is the canonical
  reference today. Other language SDKs land in subsequent waves.
</Info>

| Language    | Package                                      | Status         |
| ----------- | -------------------------------------------- | -------------- |
| TypeScript  | `@ligate-labs/sdk@rc`                        | RC, live       |
| Python      | `ligate-sdk`                                 | Planned        |
| Rust        | `ligate-sdk` crate                           | Planned        |
| Go          | `github.com/ligate-io/ligate-go`             | Planned        |

The TypeScript SDK is browser, Node, and edge-runtime compatible. Build,
sign, submit, and wait for inclusion in one call.

```bash
pnpm add @ligate-labs/sdk@rc
```

[github.com/ligate-io/ligate-js](https://github.com/ligate-io/ligate-js).
