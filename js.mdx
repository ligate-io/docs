```bash
pnpm add @ligate-labs/sdk@rc
```

```ts
import { Ligate } from '@ligate-labs/sdk'

const ligate = new Ligate({ rpc: 'https://rpc.devnet.ligate.io' })

const tx = await ligate.submitAttestation({
  schemaId: 'lsc1f8a2c…',
  payloadHash: 'lph1c3a87e…',
  attestorSig: '<BLS aggregate>',
})

await tx.waitForInclusion()
```

See [the chain RPC reference](/rpc) for the underlying surface this
wraps.
