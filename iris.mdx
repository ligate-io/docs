Iris is a Model Context Protocol server plus a USD-billed relayer for
autonomous AI agents. Agents don't hold wallets or `$LGT`. The relayer
signs and submits attestations on their behalf and bills their parent
org in USD.

Full developer docs (MCP tools, relayer pricing, agent framework
integrations) land in the next migration pass.
